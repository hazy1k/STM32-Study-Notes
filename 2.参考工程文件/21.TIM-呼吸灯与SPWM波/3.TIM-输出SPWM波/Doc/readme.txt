【*】程序简介

-工程名称：输出SPWM波
-实验平台: 野火STM32-F103 指南者 开发板 
-ST固件库版本：3.5.0

【 ！】功能简介：
使用定时器输出控制SPWM波控制LED灯

【*】注意事项：
若移植本程序到其它硬件上时，请注意要选用定时器通道

【 ！】实验操作：
编译并下载程序到开发板，复位后可以看到RGB灯呈类似呼吸效果

【*】 引脚分配

RGB灯：
	R_LED	<--->PB5(TIM3 CH2)
	G_LED	<--->PB0(TIM3 CH3)
	B_LED	<--->PB1(TIM3 CH4)

【*】 程序描述
		
		全彩呼吸灯例程和输出 SPWM波实验的工程基本一样，只是控制使用的的 PWM 表不同，
		一个为呼吸特性曲线，另一个为正弦半波曲线
