【*】 程序简介 
-工程名称：AD5689-双通道电压输出
-实验平台: 野火STM32 H750 开发板
-MDK版本：5.30
-HAL库版本：1.2.0


【 ！】功能简介：
使用开发板SPI外设控制AD5689模块输出指定电压

【 ！！】注意事项：
AD5689模块跳线帽连接：J3(RST-GND) J4(A_IN-5V) J6(GAIN-GND) J7( )

跳线帽说明：
J3 -> 上电复位选择。将该引脚连接至GND时，可将两个DAC上电至零电平。将该引脚连接至5V时，可将两个DAC上电至中间电平。
J4 -> A_OUT地选择。
J6 -> 增益选择。当该引脚与GND相连时，两个DAC的输出范围均为0 V至VREF。如果该引脚与5V相连，则两个DAC的输出范围为0 V至2 × VREF。
J7 -> B_OUT地选择。
/*********************************************************************************************/

【*】 引脚分配

开发板<--->AD5689模块

  	5V<--->5V
	SDIN<--->PF9
	 SDO<--->PF8
	SLCK<--->PF7	
	SYNC<--->PF6
	 GND<--->GND
