WIFI透传实验
1.打开野火多功能调试助手
2.在调试助手里面：打开串口调试助手、网络调试助手，设置要连接的服务器的 IP 和端口
3.在网络调试助手或者串口调试助手  发送以下命令控制板载RGB灯

  LED_RED
  LED_GREEN
  LED_BLUE
  LED_YELLOW
  LED_PURPLE
  LED_CYAN
  LED_WHITE
  LED_RGBOFF
