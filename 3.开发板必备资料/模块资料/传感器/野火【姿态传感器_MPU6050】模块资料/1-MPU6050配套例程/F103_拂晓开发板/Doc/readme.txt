【*】 引脚分配

MPU6050 使用杜邦线接到开发板：

		SCL     <--->   PD10
		SDA     <--->   PD11
        INT     <--->   该引脚不用接
		
		VCC     <--->   接5V或3.3V
		GND     <--->   接地线
		


调试串口(TTL-USB TO USART)：
CH340的收发引脚与STM32的发收引脚相连。
	RX<--->PB6
	TX<--->PB7
