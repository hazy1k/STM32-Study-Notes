【*】程序简介

-工程名称：RC522模块
-实验平台: 野火STM32 F103 开发板 
-MDK版本：5.23
-ST固件库版本：3.5.1

【 ！】功能简介：

读取ID卡

【*】注意事项：

RC522模块只能在有效区域才能读取ID卡号。

/*******************************************************************************************************************/

【*】 引脚分配

传感器模块：
STM32开发板  <---->  RC522模块
                
         3.3V<---->  VCC
         PB8 <---->  RST
         GND <---->  GND
         PB14<---->  MISO
         PB15<---->  MOSI
         PB13<---->  SCK
         PA4 <---->  SDA
