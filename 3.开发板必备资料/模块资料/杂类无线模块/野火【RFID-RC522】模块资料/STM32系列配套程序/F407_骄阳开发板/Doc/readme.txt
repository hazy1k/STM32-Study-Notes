【*】程序简介

-工程名称：RC522模块
-实验平台: 野火  STM32 F407 骄阳 开发板  
-MDK版本：5.26
-软件包：CubeF4 Firmware Package V1.25.0 


【 ！】功能简介：

读取ID卡

【*】注意事项：

RC522模块只能在有效区域才能读取ID卡号。
RC522模块检测原理是发生磁场变化，如果将ID卡放到传感器模块上再上电时，会检测不到ID卡，因为没有发生磁场变化产生中断。
RC522的中断操作需要读ComIrqReg寄存器，发送一次寻卡命令会触发一次TxIRq中断，寻卡成功一次会触发一次RxIRq中断，
          中断详细内容请参考MFRC522中文数据手册第9页。
