【*】 程序简介 
-工程名称：步进电机-S形加减速实现
-实验平台: 野火STM32 F103 拂晓 开发板
-MDK版本：5.30
-HAL库版本：1.1.4


【 ！】功能简介：
使用STM32的通用定时器定时，控制步进电机正反转和启停。



【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
步进电机驱动器使用32细分

/*********************************************************************************************/

步进电机驱动器接口1:
	PUL+<--->VCOM
	PUL-<--->PC6
	DIR+<--->VCOM
	DIR-<--->PB14
	ENA+<--->VCOM
	ENA-<--->PB12	
	
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：8MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB = 72MHz
-APB2 = 72MHz 
-APB1 = 36MHz
