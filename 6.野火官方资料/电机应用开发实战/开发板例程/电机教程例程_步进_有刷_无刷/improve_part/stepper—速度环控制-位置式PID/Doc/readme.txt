【*】 程序简介 
-工程名称：步进电机编码器测速
-实验平台: 野火STM32 F103 指南者 开发板



【 ！】功能简介：
使用STM32的通用定时器定时，控制步进电机正反转和启停。使用通用定时器编码器模式来检测电机转速。
学习目的：学习通用定时器编码器模式的配置和使用。


【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
步进电机驱动器使用32细分

/*********************************************************************************************/

步进电机驱动器:
	PUL+<--->3V3
	PUL-<--->PC6	
	DIR+<--->3V3
	DIR-<--->PB12
	ENA+<--->3V3
	ENA-<--->PB14
	
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：8MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB = 72MHz
-APB2 = 72MHz 
-APB1 = 36MHz
